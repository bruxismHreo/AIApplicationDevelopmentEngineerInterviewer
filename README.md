## 🎯 学习路径

| 阶段 | 学习方向 | 重点内容 |

| 1️⃣ | 大模型基础 | Transformer、注意力机制、预训练/微调、Tokenization |

## 📖 费曼学习法流程

1. **提问** - 面试官提出面试题
2. **讲解** - 尝试用简单语言解释（像教初学者一样）
3. **点评** - 面试官指出理解盲区
4. **深化** - 补充核心知识点
5. **记录** - 同步更新学习日志和笔记
